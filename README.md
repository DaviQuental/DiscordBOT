## 🤖 Discord Bot

- ✅ FINISHED
- This Discord Bot send timed messages to a Channel.
- Deployed on Heroku.

## 🚀 Technologies

- NodeJS
- JavaScript

## ➕ Dependencies

- Discord.js
- DotEnv

## ✂️ Clone Application

- `git clone https://github.com/DaviQuental/DiscordBOT`
- `cd DiscordBOT`

## ⚙️ Run Application

- `npm install`
- `node .`
